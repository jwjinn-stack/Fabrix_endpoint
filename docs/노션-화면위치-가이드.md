# FABRIX — 기능군별 화면·URL 위치 가이드 (노션 상세 페이지/스크린샷용)

> 노션 `🗂️ 기능 로드맵 (기능군별)` 의 각 행(기능군) 상세 페이지에 넣을 **"어느 화면 · 어느 URL · 어디에서 보이는지"** 설명입니다.
> `v0.16.1` 부터 화면별 **고유 URL(경로 라우팅)** 이 생겼습니다 — 딥링크·북마크 가능. 좌측 nav 클릭 또는 URL 직접 입력 모두 동작(브라우저 뒤로/앞으로 지원).
> URL 표기: `{BASE}` = 접속 주소(dev `http://localhost:5173`, 운영 `https://<배포도메인>`). 예: `{BASE}/usage`.

## 화면 ↔ URL 전체 맵
| 화면 | URL | 좌측 nav |
|---|---|---|
| 관제 대시보드 | `/dashboard` (루트 `/` 동일) | 관제 |
| 사용량 리포트 | `/usage` | 사용량 |
| 가드레일 | `/guard` | 가드레일 |
| 모델 레지스트리 | `/models` | 모델 |
| 모델 임포트 | `/models/import` | 모델 › 모델 임포트 |
| 플레이그라운드 | `/playground` (`?model=<id>`) | 플레이그라운드 |
| 평가 | `/eval` | 평가 |
| 엔드포인트 | `/endpoints` | 엔드포인트 |
| GPU/MIG | `/gpu` | GPU/MIG |
| 키·앱 | `/keys` | 키·앱 |
| 트래픽/프록시 | `/traffic` | 트래픽 |

## 기능군별 위치 설명 (노션 상세 페이지에 붙여넣기용)

### 1) 관제 대시보드 & 시각화
- **화면/URL**: 관제 대시보드 — `{BASE}/dashboard`
- **화면 내 위치**: 상단 KPI 4카드(실시간 트래픽·응답 품질·가드레일·GPU) → 부서/앱 분포(Top6) → 시계열(QPS·TTFT·차단) → 알람. 우상단 *기간 셀렉터·뷰 편집·새로고침*. 상단바 ☾(다크모드)·🔔(알림 드로어).
- **데이터**: VictoriaMetrics(메트릭) + ClickHouse(분포·가드 요약).
- **스크린샷 추천**: 대시보드 전체 1컷 + 🔔 알림 드로어 1컷.

### 2) 추론 성능 & 사용량 분석
- **화면/URL**: 사용량 리포트 — `{BASE}/usage`
- **화면 내 위치**: 3 KPI(요청·토큰·활성) → 점유율 스택바 → **추세 forecast** 차트 → **추론 지연 분해(TTFT/TPOT/E2E p50·p95·p99)** + **엔진 스케줄러** → **토큰 분해(입력/캐시/출력)** → **Top5 엔드포인트/API키** → 사용량 테이블. 우상단 *group_by(모델/부서/앱/키)·기간·CSV 내보내기*.
- **데이터**: ClickHouse `usage_rollup` + VictoriaMetrics(지연/스케줄러).
- **스크린샷 추천**: 추론 지연 분해+스케줄러 1컷, Top5 1컷, forecast 1컷.

### 3) GPU·트래픽 인프라 관제
- **화면/URL**: GPU/MIG — `{BASE}/gpu` · 트래픽/프록시 — `{BASE}/traffic`
- **화면 내 위치(GPU)**: 6 KPI(총GPU·평균사용률·메모리·전력·MIG효율·**유휴 할당 갭**) → 노드 LED 그리드 → GPU 테이블 → **행 클릭 시 우측 드릴다운 슬라이드(util/mem/temp/power 60분 시계열 + MIG 상태)**.
- **화면 내 위치(트래픽)**: 추론 파이프라인 다이어그램 → **엔진 파이프라인 분해(queue→prefill→decode, Tree/Waterfall 토글)** → 5 KPI → 최근 요청 스트림.
- **데이터**: VictoriaMetrics(DCGM) / proxystats·dynamo 메트릭.
- **스크린샷 추천**: GPU 드릴다운 슬라이드 1컷, 트래픽 파이프라인 분해 1컷.

### 4) 가드레일 거버넌스 & 증적
- **화면/URL**: 가드레일 — `{BASE}/guard`
- **화면 내 위치**: 상단 **탭(개요 / 증적 / 정책)**. *증적* 탭 = 5 요약카드 + 시간대별 히스토그램 + 테이블(판정·**HTTP status 403/200·지연**·정책버전) → 행 상세. *정책* 탭 = PII/Jailbreak/Secrets 토글. 우상단 분류 테스트.
- **데이터**: ClickHouse `guard_audit` + Semantic Router + MinIO(WORM).
- **스크린샷 추천**: 증적 탭(status 컬럼 보이게) 1컷, 정책 탭 1컷.

### 5) 모델 레지스트리 & 임포트
- **화면/URL**: 모델 레지스트리 — `{BASE}/models` · 모델 임포트 — `{BASE}/models/import`
- **화면 내 위치(모델)**: Harbor 모델 카드 그리드(프로젝트·태그·크기·pull·**운영 메트릭 칩**/미배포 배지) + 검색 + "+ 모델 임포트". 카드 → 엔드포인트 생성/상세.
- **화면 내 위치(임포트)**: NGC/Hugging Face/직접 업로드 소스 카드 → 모델ID 입력 → 미리보기(잡 매니페스트) → 임포트 실행.
- **데이터**: Harbor(레지스트리) + k8s Import Job.
- **스크린샷 추천**: 모델 카드 그리드 1컷, 임포트 소스 카드 1컷.

### 6) 엔드포인트 배포 & 운영
- **화면/URL**: 엔드포인트 — `{BASE}/endpoints`
- **화면 내 위치**: 배포 목록 테이블(이름·ns·백엔드·replica·상태·관리주체) + 행별 *로그·키 발급·삭제* 버튼. "+ 엔드포인트 생성" → **위저드(프리셋·패턴·Harbor 참조·접근방식·미리보기 dry-run)**. *로그* → **실시간 Pod 로그 모달**(컴포넌트 필터·자동 새로고침).
- **데이터**: Kubernetes(DynamoGraphDeployment CR·Pod 로그).
- **스크린샷 추천**: 생성 위저드(미리보기 검증) 1컷, 실시간 로그 모달 1컷.

### 7) 플레이그라운드 & 평가
- **화면/URL**: 플레이그라운드 — `{BASE}/playground` (모델 지정 시 `?model=<id>`) · 평가 — `{BASE}/eval`
- **화면 내 위치(플그)**: 좌측 설정 패널(모델·max_tokens·temperature·`</> 코드 보기`) + 우측 채팅(**응답별 지연·tok/s·토큰 칩**·가드 배지) + "모델 비교".
- **화면 내 위치(평가)**: 대상/심판 모델 + 프롬프트 → LLM-as-judge 점수·근거.
- **데이터**: Dynamo/vLLM `/v1` (가드레일 통과 후).
- **스크린샷 추천**: 채팅 응답(메트릭 칩) 1컷, 모델 비교 모달 1컷.

### 8) 키·앱 & 예산 통제
- **화면/URL**: 키·앱 — `{BASE}/keys`
- **화면 내 위치**: 키 테이블(앱·부서·쿼터 rpm/tpd·**추정비용·일 예산 사용 게이지**·상태) + "+ 키 발급" → **예산 폼(rpm·일 토큰 예산·경고 임계·리셋)**. 행 클릭 → 상세(평문 1회 노출은 발급 직후).
- **데이터**: PostgreSQL(키) + ClickHouse(사용량) + 인메모리(예산 카운터·하드캡 429).
- **스크린샷 추천**: 키 테이블(예산 게이지) 1컷, 발급 모달(예산 폼) 1컷.

### 9) 조직·RBAC & 자격증명
- **화면/URL**: 설정(RBAC) — `{BASE}/settings` · 자격증명 — `{BASE}/settings/credentials`
- **화면 내 위치(설정)**: 사용자 테이블·역할(Admin/User/Super)·부서 인라인 변경·추가/삭제. *조직·귀속(부서→앱→키)* 은 키·앱/엔드포인트 화면에 녹임(귀속 점검 중).
- **화면 내 위치(자격증명)**: HF Model Hub 토큰 / NVIDIA NGC 키 등록·수정(값 마스킹).
- **데이터**: PostgreSQL(app_user) / k8s Secret(fabrix-thirdparty).
- **스크린샷 추천**: 사용자 RBAC 테이블 1컷, 자격증명 화면 1컷.

### 10) 플랫폼 기반 & 배포
- **화면/URL**: 전역 상단바(테마·프로젝트·알림) — 모든 화면 공통. 헬스 API `{BASE}/api/v1/healthz`. (인클러스터 배포·CI/GHCR·gemma 임포트 보류는 화면이 아닌 인프라/매니페스트 — [인프라 문서](인프라-통신-아키텍처.md) 참고)
- **스크린샷 추천**: 다크모드 토글 전/후 1컷.
